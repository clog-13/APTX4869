# 1297. 子串的最大出现次数

 给你一个字符串 `s` ，请你返回满足以下条件且**出现次数 最大的 任意子串的 出现次数**：

- 子串中不同字母的数目必须小于等于 `maxLetters` 。
- 子串的长度必须大于等于 `minSize` 且小于等于 `maxSize` 。

**示例：**
```
输入：s = "aababcaab", maxLetters = 2, minSize = 3, maxSize = 4
输出：2
解释：子串 "aab" 在原字符串中出现了 2 次。
它满足所有的要求：2 个不同的字母，长度为 3 （在 minSize 和 maxSize 范围内）。
```



## 字典树

```java
class Solution {
    public int maxFreq(String s, int maxLetters, int minSize, int maxSize) {
        int N = s.length();
        int[] arr = new int[N];
        for (int i = 0; i < N; i++) arr[i] = s.charAt(i)-'a';

        TreeNode root = new TreeNode();
        for (int i = 0; i <= N-minSize; i++) {
            int state = 0, cnt = 0;
            TreeNode node = root;
            for (int j = 0; j < minSize; j++) {
                int pre = state;
                int offset = arr[i+j];
                state |= (1<<offset);
                if (state != pre) if (++cnt > maxLetters) break;
                if (node.childs == null) node.childs = new TreeNode[26];
                if (node.childs[offset] == null)
                    node.childs[offset] = new TreeNode();
                node = node.childs[offset];                
                if (j == minSize-1) node.val++;
            }
        }
        return maxFreq(root);
    }

    int maxFreq(TreeNode root) {
        if (root == null) return 0;
        int res = root.val;
        if (root.childs == null) return res;
        for (int i = 0; i < 26; i++) {
            res = Math.max(res, maxFreq(root.childs[i]));
        }
        return res;
    } 

    private static class TreeNode {
        int val;
        TreeNode[] childs;
    }
}
```

## 枚舉

```java
class Solution {
    public int maxFreq(String s, int maxLetters, int minSize, int maxSize) {
        int res = 0;
        Map<String,Integer> map = new HashMap<>();
        for (int i = 0; i <= s.length()-minSize; i++) {
            String t = s.substring(i, i+minSize);
            map.put(t, map.getOrDefault(t, 0) + 1);
        }
        for (String t : map.keySet()) {
            if (map.get(t) > res && checkMaxLetter(t, maxLetters)) 
                res = map.get(t);
        }
        return res;
    }
    private boolean checkMaxLetter(String s, int maxLetter) {
        Map<Character, Integer> map = new HashMap<>();
        char[] arr = s.toCharArray();
        for (char c : arr) {
            map.put(c, map.getOrDefault(c, 0) + 1);
            if (map.size() > maxLetter) return false;
        }
        return true;
    }
}
```

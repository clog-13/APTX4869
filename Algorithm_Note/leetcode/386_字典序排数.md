# 386. 字典序排数

给定一个整数 *n*, 返回从 *1* 到 *n* 的字典顺序。

**示例**

*n* =13，返回 [1,10,11,12,13,2,3,4,5,6,7,8,9] 。



## DFS + 剪枝

```java
class Solution {
    List<Integer> res = new ArrayList<>();
    int bit_size = 0;
    public List<Integer> lexicalOrder(int n) {
        int t = n;
		while (t != 0) {
			bit_size++;
			t /= 10;
		}
        for (int i = 1; i < 10; i++) {
            if (i > n) break;
            dfs(i, 1, n);
        }
        return res;
    }
    
    private void dfs(int i, int c, int n) {
        if (c > bit_size) return;
        res.add(i);
        for (int j = 0; j <= 9; j++) {
            int nex = i*10+j;
            if (nex > n) return;
            dfs(nex, c+1, n);
        }
    }
}
```
